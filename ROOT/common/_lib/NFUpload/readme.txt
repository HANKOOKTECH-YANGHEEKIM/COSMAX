

제품명 : NFUpload
라이센스 : 프리웨어
저작툴 : FLEX 2
제작사 : 패스코리아넷 (http://passkorea.net/)
버  전 : 1.2 (2008-10-28)

자료설명
	 NFupload 는 플래시 기반의 업로드 프로그래스바가 지원되는 멀티업로드 프로그램입니다.

사용제한
	개인이나 회사등에서 자유롭게 사용 가능합니다.
	단, 재배포시에는 본 문서를 반드시 포함하시고 출처를 밝혀주셔야 합니다.

장점
	-플래시가 동작하는 모든 OS의 웹브라우저에 지원됨 (윈도우, 매킨토시, 리눅스등)
	-별도의 ActiveX 설치가 필요하지 않으므로 접근성이 뛰어남

기능
	-HTTP 방식 업로드
	-멀티파일 업로드 기능제공
	-파일형식 필터기능제공
		 (확장자 지정업로드, 특정 확장자 업로드 차단기능  .ASP  , .PHP차단등)
	-업로드폼 크기 지정기능
	-파일갯수 및 업로드 용량제한 기능 제공
	-ASP, 닷넷, PHP 예제파일 제공
	-Flash Player 10 지원

비정기적으로 제품 기능이 변경되거나 추가될 수 있으므로 제작사 홈페이지의 공지사항을 참고 하십시오.
